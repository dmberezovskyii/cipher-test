Introduction
============

Welcome to the Secure Test Automation documentation! This project is designed to help improve security in automation frameworks. The framework is built on Python and provides encryption tools and works with various repositories to manage encryption keys.

This documentation will guide you through setting up the framework, using its features, and contributing to the project.

For more details, you can visit the project's repository on GitHub:
`Secure Test Automation Repository <https://github.com/dmberezovskyii/secure-test-automation>`_
